# CookingSite

Сайт рецептов с поиском по имеющимся продуктам.

## Стек

- **Frontend:** Next.js + TypeScript + React
- **Backend:** FastAPI + Python
- **Database:** PostgreSQL
- **AI:** OpenAI API
- **Parser:** BeautifulSoup4

## Структура

```
frontend/     - Next.js приложение
backend/      - FastAPI сервер
parser/       - Парсер рецептов
database/     - Миграции
```

## Фичи

- Поиск блюд по продуктам
- Парсинг рецептов
- Умный поиск (синонимы)
- Распознавание фото
- Личный холодильник
- Замены ингредиентов
